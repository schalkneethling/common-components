Ada Lovelace

1815-12-10
  • Considered the world's first computer programmer
  • Worked on Charles Babbage's early mechanical general-purpose computer
  • Her notes on the Analytical Engine include what is now recognized as the first algorithm intended to be carried out by a machine

Grace Hopper

1906-12-09
  • Developed the first compiler for a programming language
  • Contributed to the development of COBOL, one of the first high-level programming languages
  • Advocated for the inclusion of women in the computing field
